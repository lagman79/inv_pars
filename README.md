# AI Invoice Extractor 🧾🤖

Μια έξυπνη εφαρμογή Python που χρησιμοποιεί το Google Gemini AI για την αυτόματη εξαγωγή δεδομένων από τιμολόγια (PDF/Εικόνες) και την καταχώρησή τους σε Excel.

## Δυνατότητες
* **Αναγνώριση:** Ημερομηνία, Προμηθευτής, ΑΦΜ, Ποσά, Είδος Παραστατικού.
* **AI Vision:** Διαβάζει PDF αλλά και φωτογραφίες (JPG, PNG).
* **Smart Features:**
    * Εντοπίζει κρυμμένους Σειριακούς Αριθμούς (S/N).
    * Εντοπίζει ΜΑΡΚ/UID ακόμα και σε δύσκολα σημεία.
    * Αυτόματη κατηγοριοποίηση δαπανών (Hardware, Software, Services κ.λπ.).
* **GUI:** Εύχρηστο γραφικό περιβάλλον.

## Εγκατάσταση
1.  Install dependencies: `pip install -r requirements.txt`
2.  Run the app: `python invoice_gui.py`
